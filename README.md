# Esp32-Telegram-Welcome-Bot

Make A Telegram Welcome Bot With Esp32 

## The Commands Are :--

+ `/start` To Start The Bot
+ `/How_Are_You`
+ `/Who_Made_You`
+ `/What_Can_I_Call_You`
+ `/Sing_A_Song`
+ `/Tell_Me_A_Joke\`
